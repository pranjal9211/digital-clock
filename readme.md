# digital-clock 
This project is a simple and stylish clock display implemented using HTML, CSS, and JavaScript. The clock features a unique design with a glowing LED indicator, and the time and date are presented in a clean and visually appealing format. 

## Features: 
- Real-time clock display 
- Hover effects for an interactive user experience 
- LED indicator that glows with a pulsating animation 
- Responsive design for various screen sizes 

## Technologies Used: 
- HTML 
- CSS with Google Fonts integration 
- JavaScript for dynamic behavior 

## How to Use: 
1. Clone the repository to your local machine. 
2. Open the 'index.html' file in your web browser to view the clock. 
3. Hover over the clock to experience the interactive effects.
